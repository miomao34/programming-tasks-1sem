7.3 Каждая строка таблицы - это сведения о некоторых деталях:
деталь - количество - место хранения
Создать хеш-таблицу из списков деталей при условии,
что каждый тип деталей в одной строке снабжается ключом из названия детали.
Методы: вставить новую деталь на правильное место таблицы,
удалить деталь из таблицы, найти деталь по ключу,
определить число сравнений для каждого из поисков.
Строку таблицы отсортировать по количеству деталей в каждом месте хранения методом вставки.
Предусмотреть вывод таблицы целиком и любого из ее фрагментов.
Сохранение таблицы обеспечить в файле, чтобы начальное создание таблицы заполнять только один раз.

7.6 На каждый рейс самолета имеется список пассажиров. 
номер рейса - ФИО - Дата, время - пункт назначения
  Создать таблицу со списками всех пассажиров на одну дату.
В качестве ключа в таблице использовать номер рейса.
В каждой строке таблицы список или последовательность пассажиров одного  рейса.
Выполнить сортировку строк таблицы методом  вставки.
Обеспечить поиск  любого пассажира, получение и сохранение таблицы в файле, вывод всех сведений на экран.
Обработку всех сведений выполнять через меню.
